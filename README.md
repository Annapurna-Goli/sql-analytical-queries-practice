# 📊 SQL Practice Portfolio – From Basics to Analytical Queries

## 🚀 Overview

This repository showcases my structured SQL learning journey — starting from fundamental concepts to advanced analytical queries using Window Functions.

The focus of this project is not just writing queries, but building strong logical thinking and solving real-world data problems.

---

This project reflects my transition from writing basic SQL queries to solving analytical data problems.

## 📁 Project Structure

* `FULL_SQL_PRACTICE.sql` → Complete end-to-end practice
* `01_joins.sql` → All types of joins
* `02_groupby.sql` → Aggregations with GROUP BY & HAVING
* `03_subqueries.sql` → Basic and advanced subqueries
* `04_window_functions.sql` → Ranking, LAG, LEAD, analytical queries
* `05_case_statement.sql` → Case with window function
* `06_analytical_query.sql` → Real-world analytical SQL problem solving
* `07_cte.sql` → Common Table Expressions (CTE) with analytical filtering

## 🧠 Skills & Concepts Covered

### 🔹 Data Retrieval & Joins

* INNER, LEFT, RIGHT, FULL OUTER, CROSS JOIN
* SELF JOIN for hierarchical relationships

### 🔹 Aggregations

* COUNT, SUM, AVG, MIN, MAX
* GROUP BY & HAVING for filtering aggregated data

### 🔹 Subqueries

* Nested queries
* Filtering using subqueries
* Aggregation inside subqueries

### 🔹 Window Functions (Analytical SQL)

* DENSE_RANK(), ROW_NUMBER()
* LAG() & LEAD() for row comparison
* PARTITION BY for grouping without collapsing data
* Running totals and cumulative calculations

### 🔹 Conditional Logic

* CASE statements
* Above/Below average classification

### 🔹 Analytical SQL Queries

* Teacher-wise student ranking
* Comparing overall average vs group average
* Top performers analysis
* Performance bucket classification
* Multi-condition analytical filtering
* Real-world reporting logic using SQL

### 🔹 Common Table Expressions (CTE)

* Query readability improvement
* Step-by-step analytical query building
* Reusable temporary result sets
* Combining CTE with Window Functions & CASE
